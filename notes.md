# Notes
**Terms:**
* Git - snapshot of folder
* Branch - copy of folder
* Fork - copying to another account
* Pull - save to your machine
* Repository - folder

**Markdown:**
* bold - double asterisks
* italics - single asterisks
* heading - #
* bullets - asterisk
* numbered list - 1. 
* link - `[text](link)`
* image - `![alt text](link)`
* block quote - >
* inline code - `
